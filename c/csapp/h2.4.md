# homework 2.4 hex add minus

## question
1. 0x503c + 0x8    = ?
2. 0x503c - 0x40   = ?
3. 0x503c + 65     = ?
4. 0x50ea - 0x503c = ?

## answer

1. 0x503c + 0x8    = 0x5044
2. 0x503c - 0x40   = 0x4FFC
3. 0x503c + 65     = 0x507D
4. 0x50ea - 0x503c = 0xAE
