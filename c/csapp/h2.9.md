# homework 2.9 color

## question

R G B|color|R G B|color
---  |---  |---  |---
0 0 0|black|1 0 0|red
0 0 1|blue |1 0 1|purple
0 1 0|green|1 1 0|yellow
0 1 1|cyan |1 1 1|white

blue   | green  = ?  
yellow & cyan   = ?  
red    ^ purple = ?  

## answer

blue   | green  = 001 | 010 = 011 = cyan  
yellow & cyan   = 110 & 011 = 111 = white  
red    ^ purple = 100 ^ 101 = 001 = blue
